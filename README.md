# 开放同行评议与高水平论文

这是一个静态研究展示站点，用于呈现开放同行评议数据与高水平论文识别分析。页面将总集与三个代表性领域子集放在同一套可比较的网页结构中。

## 页面入口

- `index.html`：总集页面
- `ai.html`：人工智能子集页面
- `biomedicine.html`：生物医学子集页面
- `physics.html`：物理学子集页面

四个页面共用同一套脚本、样式和图表说明逻辑，通过 `body[data-dataset]` 切换数据集内容。

## 内容结构

每个页面包含 8 个章节：

1. 第一章 引言
2. 第二章 数据采集与构建
3. 第三章 特征工程
4. 第四章 描述性分析
5. 第五章 变量相关性分析
6. 第六章 模型构建与结果
7. 第七章 总结与讨论
8. 第八章 项目成果与开放资源

页面首屏统一展示四项数据概况：`记录数`、`唯一论文数`、`变量字段数`、`图表数量`。首屏下方的三张说明卡统一为：`数据来源`、`质量概况`、`时间结构`。

## 图表与模型

- 每个数据集页面展示 83 张图。
- 图表资源位于 `assets/datasets/`：
  - `overall/`
  - `ai/`
  - `biomedicine/`
  - `physics/`
- `visual-descriptions.js` 为每个数据集提供逐图解读。
- 第六章按当前页面展示对应数据集的引用 Top10 预测模型表，包含 XGBoost、LightGBM 和随机森林。

## 文件说明

- `site.js`：页面数据、章节内容、首屏概况、模型表和客户端渲染逻辑。
- `site.css`：基础布局与视觉样式。
- `visual-overrides.css`：补充样式，包括首屏卡片、章节说明、模型表和左侧目录当前章节高亮。
- `visual-descriptions.js`：按数据集替换图表解读文字。
- `copy-cleanup.js`：页面文案清理逻辑。
- `AGENTS.md`：项目维护约定。

## 交互特性

- 顶部导航可在总集、人工智能、生物医学和物理学页面之间切换。
- 左侧目录可跳转到对应章节。
- 页面滚动时，左侧目录会自动高亮当前章节。
- 页面可直接通过静态文件打开；使用本地 HTTP 服务预览也可以。

## 本地查看

直接打开任一入口文件即可，例如：

```text
index.html
ai.html
biomedicine.html
physics.html
```

如需通过本地服务预览：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/
```

## 维护注意

- 新增或替换图表时，保持四个数据集目录中的文件名一致。
- 修改章节结构时，同步更新 `site.js` 中的章节映射和左侧目录逻辑。
- 修改首屏数据概况时，四个页面应保持同一组字段标签。
- 不要把运行时生成文件、系统文件或无关本地文件提交进仓库。
